# Git++
The Git version control system rewritten in C++

> [!NOTE]
> Please note that Git++ is still under **heavy** development and is not ready for use.

# Why Git++
Git++ is written purely in C++ which is an improved variant of the C programming language. Git++ also focuses on being as minimal as possible, without using many different programming languages for its functionality unlike original Git.
